#MM...FOOD 

* AS A food enthusiast
* I WANT food inspiration, and recipes to come along with it
* SO THAT i can eat!!

* GIVEN a simple website, with a button that says, "mm... food"
* WHEN i push the mysterious button
* THEN a picture of a randomly generated, scrumptious food dish appears on the screen
* WHEN i look beneath the picture
* THEN i see mouth watering recipes displayed at my very fingertips
* WHEN i tap or click on one of these recipes
* THEN the recipe appears full screen, so i can read it to completion

In this project we first strated by getting first server and retireving random images. Ater getting that image we created an image element in html. Sotred that random picture in that image element. Then stored that image name into an varibale so it can be used a paramter for the next server whuich will give us our reciepe for the dish shown in image element. 

In the second server we passed 4 required parameters like food name, app id, key and either public or private status, after that we stored the reciepe from data into a varibale and displayed it on the webpage. We created a button and added an event listener to it and gace it a fucntion which call getImage() function which will show the dish and recipe upon clicking on it.
